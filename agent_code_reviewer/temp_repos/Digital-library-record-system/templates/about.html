Digital Library Records System

Digital Library Records System is a simple and efficient web developed to manage book records, student details, issue dates, return dates and department-wise book tracking. It helps automate the manual process used in libraries and makes data handling faster.

Purpose of the Project

  • To maintain student book issue and return records digitally.
  • To reduce manual paperwork in the library.
  • To store, update, search and delete records easily.
  • To help librarians manage books efficiently.

Features

  • Add new book issue records
  • Update existing book details
  • Delete incorrect/old records
  • Display all student book entry records
  • Simple and user-friendly interface

Technologies Used

  • Frontend: HTML, CSS, Bootstrap
  • Backend: Flask (Python)
  • Database: MySQL (SQLAlchemy ORM)

This project is mainly created for learning Flask, CRUD operations and database connectivity. It is suitable for college mini projects and portfolio building.
